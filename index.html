Projects

  • The Guide to Nonexistent Birds - an Ornithological Logic

    A generative fictional birdwatching guide, with an essay on taxonomy, formal logic, and the OuLiPo written in the comments to the Prolog code on the left. My first major digital writing project, some of the sentiments in the essay are a bit naive or immature, but it is also the project I feel most gleefully proud of.

  • Lost in the Asset Store

    A digital space populated with free assets for the game engine Unity. Prefab logic, free assets, and free-to-copy code enter into the great majority both technical and digital artistic production, and this project means to explore, on their own terms, those digital commons.

  • Textileexittext: a computational embroidery project exploring metaphors and histories between text, textile, and computation. Used a corpus, Python NLP, and an embroidery machine to stitch generated text.
